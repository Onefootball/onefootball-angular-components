Base setup

Installation 


bower install onefootball-angular-components


npm install onefootball-angular-components

Include

After installation, you just have to declare dependency in your module. We depend on angular-sanitize, so you should add it to your project. 


   var demoApp = angular.module('demoModule', [
        'onefootball.components',
        'ngSanitize'
    ]);
