GUID

This service generates unique user id (GUID, UUID, UID)

Generated GUID: {{GUID}}

Example code
Javascript

demoControllers.controller('serviceCtrl', ['$scope', 'ofGuid', function ($scope, ofGuid) {
    $scope.generateGuid = function () {
        $scope.GUID = ofGuid.generate();
    };
}]);
HTML

    <h3 id="guid">GUID</h3>
    <p>
        This service generates unique user id (GUID, UUID, UID)
    </p>

    <button class="btn btn-default" data-ng-click="generateGuid()">Generate</button>

    <p style="margin-top: 12px;" data-ng-if="GUID">
        <span> Generated GUID: { {GUID } } </span>
    </p>
