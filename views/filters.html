Cyrillic2latin filter

This filter transliterate cyrillic to latin. We use it for generating cyrillic urls (to use ascii urls).

  • {{:: item}} ==> {{:: item | cyrillic2latin}}

New lines filter

Often text is server from backend and has strange line breaks. This filter, together with ngSanitize is used to normalize line breaks in chunk of text (output is html).

Original text
Output
        {{::newLinesText}}
As html text
New lines text
Output
            {{::newLinesText | newlines}}
As html

Replace filter

This is a simple string replace filter. First argument is target to replace, second is replacement pattern and last one (optional) are regular expression flags.

Original string
{{::strReplace}}
Replace string (a witch c)
{{::strReplace | replace : 'a' :'c'}}
Replace string globally (a with c and flag g)
{{::strReplace | replace : 'a' :'c' : 'g'}}
Replace string globally and ignore casing (a with c and flags gi)
{{::strReplace | replace : 'a' :'c' : 'gi'}}
