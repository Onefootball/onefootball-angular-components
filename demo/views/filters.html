Cyrillic to Latin filter

This filter transliterate cyrillic to latin. We use it for generating cyrillic urls (to use ascii urls).

  • {{:: item}} ==> {{:: item | ofCyrillic2Latin}}
Example code
Javascript

        demoControllers.controller('filterCtrl', ['$scope', function ($scope) {
            $scope.cyrillicList = ['радиатор', 'Антарктика', 'телеграмма', 'Англия', 'витамин', 'мираж', 'Шанхай'];
        });
HTML

        <h3>Cyrillic2latin filter</h3>
        <p>
            This filter transliterate cyrillic to latin. We use it for generating cyrillic urls (to use ascii urls).
        </p>

        <ul>
            <li ng-repeat= "item in cyrillicList">{ {:: item} } ==> { {:: item | ofCyrillic2Latin} }</li>
        </ul>

Diacritic remove filter

This filter can be used to strip accents from words. This can be usefull, when you want to sort words alphabetically, or when you want to generate url from name.

  • {{:: item}} ==> {{:: item | ofDiacriticStrip}}
Example code
Javascript

        demoControllers.controller('filterCtrl', ['$scope', function ($scope) {
            $scope.diacriticList = ['dégagé', 'déjà vu', 'démarche', 'démodé', 'dénouement', 'Mützen', 'tête-à-tête'];
        });
HTML

        <h3>Diacritic filter</h3>
        <p>
            This filter can be used to strip accents from words. This can be usefull, when you want to sort words alphabetically,
            or when you want to generate url from name.
        </p>

        <ul>
            <li ng-repeat= "item in diacriticList">{ {:: item} } ==> { {:: item | ofDiacriticStrip} }</li>
        </ul>

Json pretty print filter

This filter can be used to quickly pretty print JS object.

Normal output
{{:: customObject}}
Pretty output
{{:: customObject | ofJsonPrettyprint}}
Example code
Javascript

        demoControllers.controller('filterCtrl', ['$scope', function ($scope) {
           $scope.customObject = {
                name: 'Sinisa',
                company: {
                    name: 'Onefootball Gmbh',
                    address: 'Greifswalder strasse 212'
                },
                status: 'Awesome',
                food: ['Steak, pizza, coffee']
            };
        );
HTML

        <h3>Json pretty print filter</h3>
        <p>
            This filter can be used to quickly pretty print JS object.
        </p>

        <h6>Normal output</h6>
        <pre>
            { {:: customObject} }
        </pre>

        <h6>Pretty output</h6>
        <pre>
            { {:: customObject | ofJsonPrettyprint} }
        </pre>

New lines filter

Often text that is served to the applications, have some strange line breaks. This filter, together with ngSanitize is used to normalize line breaks in chunk of text (output is html).

Original text
Output
{{::newLinesText}}
As html text
New lines text
Output
{{::newLinesText | ofNewLines}}
As html
Example code
Javascript

        demoControllers.controller('filterCtrl', ['$scope', function ($scope) {
          $scope.newLinesText = 'This is some text\n that has a line break, <br/> <br/> <br/> <br/> <br/> \n\n\n\n\n which is unusual!';
        });
HTML

        <h3>New lines filter</h3>
        <p>
            Often text is server from backend and has strange line breaks.
            This filter, together with ngSanitize is used to normalize line breaks in chunk of text (output is html).
        </p>

        <h5>Original text</h5>
        <h6>Output</h6>
        <pre>
            { {::newLinesText} }
        </pre>
        <h6>As html text</h6>
        <div ng-bind-html="::newLinesText">
        </div>

        <h5>New lines text</h5>
        <h6>Output</h6>
        <pre>
                { {::newLinesText | ofNewLines} }
        </pre>
        <h6>As html</h6>
        <div ng-bind-html="::newLinesText | ofNewLines">
        </div>

Order object by filter

This filter is used to order list of objects by property.

Original list
  • name: {{:: item.name}}, id: {{:: item.id}}
List of object sorted by id
  • name: {{:: item.name}}, id: {{:: item.id}}
List of object sorted by name
  • name: {{:: item.name}}, id: {{:: item.id}}
List of object sorted by name reverted
  • name: {{:: item.name}}, id: {{:: item.id}}
Example code
Javascript

        demoControllers.controller('filterCtrl', ['$scope', function ($scope) {
            $scope.listOfObjects = [
                {
                    id: 5,
                    name: 'Zan'
                },
                {
                    id: 1,
                    name: 'Masa'
                },
                {
                    id: 10,
                    name: 'Neza'
                }
            ];
        });
HTML

        <h3>Order object by filter</h3>
        <p>
            This filter is used to order list of objects by property.
        </p>

        <h5>
            Original list
        </h5>
        <ul>
            <li ng-repeat= "item in listOfObjects">
                name: { {:: item.name} },
                id: { {:: item.id} }
            </li>
        </ul>

        <h5>
            List of object sorted by id
        </h5>
        <ul>
            <li ng-repeat= "item in listOfObjects | ofOrderObjectBy : 'id'">
                name: { {:: item.name} },
                id: { {:: item.id} }
            </li>
        </ul>
        <h5>
            List of object sorted by name
        </h5>
        <ul>
            <li ng-repeat= "item in listOfObjects | ofOrderObjectBy : 'name'">
                name: { {:: item.name} },
                id: { {:: item.id} }
            </li>
        </ul>
        <h5>
            List of object sorted by name reverted
        </h5>
        <ul>
            <li ng-repeat= "item in listOfObjects | ofOrderObjectBy : 'name' : true">
                name: { {:: item.name} },
                id: { {:: item.id} }
            </li>
        </ul>

Replace filter

This is a simple string replace filter. First argument is target to replace, second is replacement pattern and last one (optional) are regular expression flags.

Original string
{{::strReplace}}
Replace string (a witch c)
{{::strReplace | ofReplace : 'a' :'c'}}
Replace string globally (a with c and flag g)
{{::strReplace | ofReplace : 'a' :'c' : 'g'}}
Replace string globally and ignore casing (a with c and flags gi)
{{::strReplace | ofReplace : 'a' :'c' : 'gi'}}
Example code
Javascript

    demoControllers.controller('filterCtrl', ['$scope', function ($scope) {
       $scope.strReplace = 'aaaaaaaAaaaaaaabbbb';
    });
HTML

        <h3 >Replace filter</h3>
        <p>
            This is a simple string replace filter. First argument is target to replace, second is replacement pattern and
            last one (optional) are regular expression flags.
        </p>

        <h5>Original string</h5>
        <div>{ {::strReplace} }</div>

        <h5>Replace string (a witch c)</h5>
        <div>{ {::strReplace | ofReplace : 'a' :'c'} }</div>

        <h5>Replace string globally (a with c and flag g)</h5>
        <div>{ {::strReplace | ofReplace : 'a' :'c' : 'g'} }</div>

        <h5>Replace string globally and ignore casing (a with c and flags gi)</h5>
        <div>{ {::strReplace | ofReplace : 'a' :'c' : 'gi'} }</div>

Url safe text filter

This filter converts random text to url save text (it supports also cyrillic text). It can be used to generate slugs, or rewrite urls.

Original text
{{:: textForUrl}}
Text "urlzied"
{{:: textForUrl | ofUrlSafe}}
Example code
Javascript

        demoControllers.controller('filterCtrl', ['$scope', function ($scope) {
           $scope.textForUrl = 'Jose Mourinho camp "frustrated" with Manchester Utd job situation - sources';
        });
HTML

        <h3 id="urlText">Url text filter</h3>
        <p>
            This filter converts random text to url save text (it supports also cyrillic text). It can be used to generate slugs,
            or rewrite urls.
        </p>

        <h5>Original text</h5>
        <div>{ {:: textForUrl} }</div>

        <h5>Text "urlzied"</h5>
        <div>{ {:: textForUrl | ofUrlSafe} }</div>
